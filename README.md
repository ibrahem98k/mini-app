# تطبيق طلبات صيانة (Front-end فقط)

واجهة موبايل عربية (RTL) مبنية بـ Svelte + Vite لطلب فنيين صيانة بمختلف التخصصات.

## المزايا
- شاشة رئيسية سريعة مع أقسام صيانة شائعة
- نموذج طلب صيانة مع الحقول الأساسية وصور (محليًا)
- قائمة "طلباتي" مع فلترة بالحالة/القسم وبحث
- تخزين محلي (localStorage) بدون باك-إند
- تصميم موبايل-أول ومتجاوب

## التشغيل محليًا
1. المتطلبات: Node.js 18+
2. تثبيت الحزم:
   ```bash
   npm install
   ```
3. تشغيل بيئة التطوير:
   ```bash
   npm run dev
   ```
4. افتح المتصفح على: http://localhost:5173

## هيكلة المجلدات
- `src/App.svelte`: الصفحة الرئيسية والتنقل بين التبويبات
- `src/components/`: المكونات (`BottomNav`, `CategoryPicker`, `RequestForm`, `RequestsList`)
- `src/stores/requests.js`: متجر الطلبات والمتزامن مع `localStorage`
- `src/lib/categories.js`: أقسام الصيانة
- `src/styles.css`: تنسيقات عامة متوافقة مع RTL

## ملاحظات
- هذا المشروع واجهة فقط. لا يوجد إرسال حقيقي للسيرفر.
- الصور تحفظ مؤقتًا داخل المتصفح بصيغة Base64 في `localStorage` (للتجارب فقط).
- لمعالجة بيانات حقيقية، اربط `stores/requests` مع API لاحقًا.
